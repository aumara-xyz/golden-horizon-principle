## Promotion Rules

- Promote only when a claim crosses a real evidentiary threshold, not because it sounds coherent.
- Mathematical promotion requires theorem-grade closure inside a stated domain.
- Physical promotion requires a defined bridge object, explicit failure condition, and stronger support than toy telemetry.
- Engineering promotion requires multi-seed stability, sensible baselines, and no silent overclaim.

## Demotion Rules

- Demote when tests fail, bridges stay undefined, or a result remains only numerical resonance.

## Recent Experiment Arc - May 2026

### Golden Zipper / observer-memory line

The week-long toy arc from `golden_zipper_v4_null_test.py` through `golden_zipper_v32e_contextual_sheath_recall.py` converged on a narrower but stronger statement:

- Exact golden uniqueness did not survive.
- Phi now reads better as anchor / anti-locking background than as direct write point.
- The observer-side picture improved when the toy shifted from raw sequence comparison to boundary, witness, revision, and later delayed / relational slot models.
- `v38` and `v38b` strengthened the move from slot identity to field-plus-binding identity: field presence plus binding outperformed earlier abstractions, but shuffled phase barely hurt, so phase itself remains only partially load-bearing.
- `v39`, `v40`, and `v41` kept the line alive: multiscale observer windows help, the combined multiscale-plus-binding model stays strong, and a soft admission band gives a small extra lift without becoming a unique explanation.
- `v41b` through `v44` tightened the line further: the strongest current variant is the prediction-error field (`v42` / `v42b`), where moderate mismatch slightly beats exact-match, novelty, and no-band admission; competition-pressure variants (`v43` / `v43b`) were comparatively weak.
- `v45` through `v47` did not overturn that lane: harsher admission-band nulls still left moderate mismatch slightly ahead, and observer-window ablation says nested windows help more than plain local windows. But when the late-stage lanes are compared by summed ablation robustness, the broader `v41b` field-stack edges out the narrower `v42` story.
- `v48` through `v50` tightened the picture again: the cleanest local rule is now “nested observer window plus moderate mismatch,” and the prediction-error benefit survives even when binding is weakened. But the carry-forward shootout still ranks the broader `v41b` field-stack above the narrower sub-panels, so the right move is to carry both a clear local rule and a broader robust package.
- `v51` through `v55` tested whether recall should literally rewrite the field a little each time. In this implementation, it did not help: rigid or less-plastic variants stayed better, and the plasticity panels did not dislodge the broader `v41b` stack. So the humane interpretation is not “plasticity is false,” but “this toy expression of plasticity is not yet the right one.”
- `v56` and `v57` pushed a fairer version of that same intuition: some memories only need a touch-up, others need to melt and resettle. Even there, the toy still preferred rigid storage, and the carry-forward shootout kept `v41b` on top. So for now the software result is: plasticity remains phenomenologically plausible, but this whole rewrite-layer family is not yet earning its keep.
- The strongest clean positives so far are not "golden wins," but "relational groove memory outperforms isolated memory" and "recall works best when context tints the same groove instead of relocating it."
