## One-Paragraph Explanation

The Golden Horizon Principle (GHP) is strongest as an observer-boundary architecture program, not as a claim that the golden ratio numerically controls all of physics. The hard mathematical spine is that Fibonacci is the minimal braiding-universal non-abelian architecture inside a stated categorical class. The open problem is whether finite observers force a further selection principle: do readable, writable, recoverable, shareable boundaries prefer a smallest non-abelian closure, and if so is Fibonacci that first closure? The project now has a live toy corridor around finite access, memory inertia, recovery, contradiction, and coherence, but it still lacks the actual write-law and the actual selection functional.

## The Biggest Open Questions

These are the questions most worth attacking.

### Q1. The write-law

Can one derive a non-ad hoc write-law from a finite observer boundary?

The toy language is:

\[
E_t : M_t \to N_t
\]

But the missing object is the rule that says when a flowing process becomes a durable record.

The actual target is some principled write-event rule, not a hand-built threshold.

### Q2. The selection functional

Can one define a functional on admissible observer-boundary architectures whose minimum is Fibonacci, or prove that this is impossible?

Very rough form:

\[
\mathcal{J}(\mathcal{B}) = \text{access cost} + \text{repair cost} - \text{recoverability} - \text{shared readability}
\]

No such functional is derived yet. Right now it is only a research shape.

### Q3. The bridge object

Can the current bridge stack be unified into one real mathematical object rather than a list of imported analogies?

Current stack:

- Markov blankets / multi-scale FEP
- conditional expectation / split property
- shared-interface geometry
- holographic recoverability
- observer-relative access kernels

The key problem is whether these can be glued into one falsifiable observer-boundary object.

### Q4. The local selector problem

In the toy corridor, the observer often faces two jobs:

- follow a coherent signal
- scrub internal contradiction

Recent toy result:

- full local switcher accuracy: about `0.890`
- compact strange-familiar switcher: about `0.837`

Question:

Can one formalize a local criterion that distinguishes:

```text
new but true
vs
internally tangled
```

without access to global truth?

### Q5. The invariant

What is the one quantity or structure that must survive bounded access, partial damage, and recovery?

Current best verbal candidate:

```text
recoverable identity-continuity under finite access
```

But it is not formal yet.

### Q6. The coherence/contradiction regime split

Recent toy work is separating two jobs:

- `fibonacci = coherence tether`
- `generic_ternary = contradiction scrubber`

Question:

Is there a rigorous invariant or stability criterion that explains why one family is favored in coherent regimes and another in internally contradictory regimes?

### Q7. Disproof challenge

The most useful negative result may be a proof that no reasonable observer-boundary functional can select Fibonacci uniquely.

That would still be a major contribution because it would collapse a fuzzy theory into a smaller, more honest one.
