# Ternary Boundary Operator Summary

## Summary

Three local toy passes were run.

- v1 passive witness failed because witness became too passive and consolidation was too weak.
- v2 active witness improved delayed or unresolved meaning handling, but results remained mixed.
- v3 hybrid selective witness worked best: Binary Ricci remained the default, while witness acted as an optional uncertainty / quarantine operator.

## Action Meanings

- Write = durable memory.
- Witness = temporary unresolved buffer.
- Release = compression / demotion / return.

## Current Interpretation

The strongest supported architecture is:

- Binary Ricci as the default memory-pressure law.
- Witness as a bounded special-case operator for delayed, contradictory, uncertain, or high-salience unresolved meaning.

## Status

Engineering heuristic / toy telemetry only.
