## Golden Zipper Research Arc

This lane moved from naive golden “winner” thinking toward more specific observer-memory questions.

### What survived

- relational memory beats isolated slot memory
- delayed or contextual re-entry matters
- region/family identity matters more than exact-slot identity
- field presence matters more than exact field shape
- multiscale observer windows help

### What weakened

- phase-specific claims
- exact golden uniqueness
- simple rewrite/plasticity-as-win stories
- frame/lens-only reinterpretation as the main mechanism

### Best current reading

The toy seems to prefer something like:

```text
memory as a relational groove or field
rather than a discrete static slot
```

and:

```text
recall as re-entering a nearby region
rather than landing on one exact old point
```

That is interesting. It is not physics evidence.

## Boundary Access Research Arc

This lane became the main live engine recently.

### First major shift

The project stopped asking:

```text
does Fibonacci win every score?
```

and started asking:

```text
what job is each structure doing?
```

That was a big improvement.

### Current role split

Safest toy summary:

```text
fibonacci = coherence tether
generic_ternary = contradiction scrubber
```

That role split became much sharper over time.

### More precise version

- missing signal -> Fibonacci
- overload signal -> Fibonacci
- coherent outside pattern -> Fibonacci
- internally scrambled wrong signal -> generic ternary

This means the story is not:

```text
ternary handles noise
```

It is more like:

```text
ternary handles internal contradiction
Fibonacci handles coherent continuation
```

### Noise/crossover result

Under one narrower class of uniform-smear helper corruption:

- low noise -> Fibonacci
- mid/high noise -> generic ternary can win

But that crossover does **not** generalize to every noise family.
Gaussian, delayed, permuted, and cross-family corruption mostly returned Fibonacci to the top.

So the honest update is:

```text
the turbulence exception is real
but narrower than it first looked
```

### Whole-stack result

When observer access cost and repair-switch cost were added:

- Fibonacci still held the normal legal-access lane
- generic ternary still only really took over in specific contradiction-heavy or narrow turbulence lanes

That made the architecture more believable, not less.

## Switcher Results

The switcher is one of the strongest recent results.

### Full local switcher

Held-out target-family accuracy:

- always `fibonacci`: about `0.253`
- always `generic_ternary`: about `0.747`
- full local switcher: about `0.890`
- oracle: `1.000`

This suggests a local observer may be able to choose the right correction geometry from boundary symptoms alone.

### Ablation result

The switcher is not standing on one magical clue.

- full mixed feature set: about `0.890`
- damage-only: about `0.866`
- no-helper-alignment: about `0.866`
- no-entropy: about `0.849`
- helper-only / entropy-only / overlap-only: all around `0.747`

So the selector seems to need several small signs together.

### Strange-familiar compression

- surprise

- strangely foreign

Results:

- four-axis compact pack: about `0.837`
- compact strange-compass: about `0.772`
- familiarity alone: about `0.747`
- surprise alone: about `0.747`

Best read:

```text
"strangely familiar" versus "strangely foreign"
is a real sub-language of the selector
but not the whole selector
```
